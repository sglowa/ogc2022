Tak jak zostal on zapowiedziany : oto format pracy zaliczeniowej:

Strona html importujaca model stworzony w blenderze.

  • Poslugujac sie narzedziami w edit mode + sculpt mode + modifiers, prosze o przygotowanie modelu.
  • Dla modelu prosze takze o przygotowanie mapy UV i tekstury ktora zostanie na niego nalozona.
  • Model eksportowac bedziemy w formacie obj. i ‘importowac’ go w obszar przegladarki, dlatego bierzmy pod uwage ilosc poligonow z ktortych sie sklada (im wieksza rozdzielczosc tym bardziej nasza przegladarka bedzie sie krztusić)
  • nastepnie : stworzyc prosta strone HTML, i zawrzec w niej Javascript importujacy stworzony model.
