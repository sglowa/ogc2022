na poczatek - pare rzeczy ktore nie zadzialaly tak jak powinny :

  • mirror modifier - tutaj troche trudno ocenic co sie dzieje, na podstawie screenu nie dam rady zdiagnozowac; jesli dalej bedziesz miala problem z mirror to sprobuj go troche dokladniej opisac i dojdziemy do tego co tam sie dzieje. generalnie, osie ktorych uzywa mirror modifier do tworzenia tych odbic przebiegaja przez origin point naszego modelu (ta mala pomaranczowa kropka). Wiec jesli nasza kropka znajduje sie w srodku obiektu to ten obiekt jest de facto duplikowany, odwrocony, i nakladany sam na siebie - a jesli ten obiekt juz byl symetryczny, tak jak ta malpa to to ze teraz jest podwojony nie bedzie widoczne. Wiec zeby troche lepiej zobaczyc jak dziala mirror modifier, proponuje poprzemieszczac troche origin point (Object -> Set Origin), albo np. usunac czesc modelu w Edit Mode . Przypomnienie: przesuwajac nasz model w edit mode polozenie jego origin point pozostaje niezmienne (origin point w ktorym mirror modifier stawia os symetrii).

  • jesli chodzi o Array modifier

    1: Array Modifier bierze dlugosc curve ktørego w nim uzywamy (tryb Fit Curve) ale nie jego polozenia czy obrotu - innymi slowy to w ktorym kierunku powielany jest nasz obiekt zalezy od ustawien Offset w samym modifierze.
    curve wydluzam wertykalnie ale obiekt kopiowany jest horyzontalnie

    2: jesli wydluzamy nasza krzywa w Object Mode musimy potem ta transformacje zaaplikowac (uzywajac Object -> Apply (ctrl/cmd + a) -> scale |albo| -> all transforms). Natomiast jesli manipujemy tym curve w edit mode , skalujac go albo ekstrudujac albo przemieszczajac jego punkty kontrolne, to wtedy to jak jego zmieniajacy sie ksztalt wplywa na Array modifier zobaczymy odrazu.

curve wydluzany w edit mode

Ciesze sie ze zwieksza sie komfort z jakim nawigujesz po programie!
jesli chodzi o same ramy tych 'zadan domowych' ( ##spotkanie_04.03>zadanie domowe & ##spotkanie_18.03>zadanie_domowe ) - to w sumie nie musisz zbierac 'dowodow' na to ze cwiczysz - bardziej chodzi o to zeby przy okazji tych ekspolarcji wejsc glebiej w konkretny zakamarek programu (konkretniej wybrany modifier(s) i sculpting tool(s)), przebadac rozne opcje ktore oferuje; poczytac troche o nim, poszperac na youtubie, zobaczyc jak zachowuje sie na roznych modelach, roznych rozdzielczosciach modeli, przy roznych ustawieniach (eg. dyntopo on/off dla scultping tools ) i opisac to w sposob ktory bedzie zrozumialy dla innych (eg. reszty osob w naszej grupie). Innymi slowy - udokumentowac go tak aby przedstawic to narzedzie w sposob w jaki ty bys chciala zeby przedstawione zostaly ci inne narzedzia :^) dodatkowo, przy troche dokladniejszym opisie latwiej bedzie diagnozowac czemu pewne rzeczy nie daja oczekiwanych rezultatow (jak w obecnym przypadku mirror modifier).
